# Умный ассистент поддержки для интернет‑магазина

> Связка Telegram‑бота (aiogram), API‑шлюза (FastAPI) и внешних HTTP‑запросов (requests). Проект показывает умение работать с асинхронностью, переменными окружения, обработкой ошибок и разделением ответственности между сервисами, бот принимает запросы от пользователей, API‑сервис координирует логику и внешние вызовы, а ответы возвращаются в удобном виде.

## Цель

Реализовать простой, но полноценный микросервисный сценарий:

- Telegram‑бот (aiogram) — интерфейс для пользователя.
- FastAPI — промежуточный API: валидация, логика, интеграция.
- Внешний сервис (через `requests`) — источник данных (в демо‑режиме заменены на готовые карточки товаров на русском).
- Обработка ошибок и понятные сообщения вместо «падения» приложения.

Проект спроектирован так, чтобы внешний источник данных (включая ИИ‑API) можно было легко заменить без переписывания всей системы.

---

## Стек технологий

- **Python 3.11+**
- **FastAPI + uvicorn** — API и сервер
- **aiogram 3.x** — Telegram‑бот
- **requests** — HTTP‑запросы к внешним API
- **python‑dotenv** — безопасное хранение токенов и URL
- **Pydantic** — валидация данных

## Быстрый старт

### Шаг 1. Клонирование

    git clone <URL_РЕПОЗИТОРИЯ>
    cd tg-project


### Шаг 2. Виртуальное окружение

Windows (cmd):

python -m venv .venv

.venv\Scripts\activate

macOS / Linux:

python3 -m venv .venv

source .venv/bin/activate

### Шаг 3. Установка зависимостей

pip install -r requirements.txt

### Шаг 4. Настройка переменных окружения

Заполните данные в .env

## Запуск

### 1. Запустите FastAPI (в первом терминале)

uvicorn main:app --reload

Сервис будет доступен по адресу: http://127.0.0.1:8000

Swagger UI (документация API): http://127.0.0.1:8000/docs

### 2. Запустите Telegram‑бота (во втором терминале)

Убедитесь, что в этом терминале тоже активировано .venv

python bot.py

## Как пользоваться

Найдите своего бота в Telegram.

Отправьте /start — получите приветствие.

Отправьте /recommend — бот запросит рекомендации у FastAPI и покажет 3 карточки товаров на русском языке.
